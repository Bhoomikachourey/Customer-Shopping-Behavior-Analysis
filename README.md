# 🛒 Customer Shopping Behavior Analysis using Python

## 📌 Project Overview

This project analyzes customer shopping behavior and demographics using Python. It explores purchasing patterns, customer demographics, and sales trends through interactive visualizations created with Pandas, Matplotlib, and Seaborn.

The analysis helps businesses identify valuable customer segments, understand purchasing behavior, and support data-driven decision-making.

---

# 📊 Dashboard 1: Customer Shopping Behavior Analysis

![Shopping Dashboard](images/customer_shopping_dashboard.png)

### Insights

- Uttar Pradesh generated the highest sales.
- Female customers contributed significantly more sales than male customers.
- Food was the highest-selling product category.
- Customers working in the IT sector generated the highest revenue.
- Single customers spent more than married customers.
- Customers aged 26–35 years contributed the highest sales.

---

# 📊 Dashboard 2: Customer Demographics Analysis

![Demographics Dashboard](images/customer_demographics_dashboard.png)

### Insights

- Female customers represent nearly 70% of all customers.
- Male customers account for approximately 30%.
- Around 58% of customers are single.
- Married customers represent approximately 42%.
- The customer base is dominated by female and single shoppers.
